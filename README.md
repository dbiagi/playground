# Playground

This repository is a playground for testing frontend little projects.

## Tech stack

I'll try to use just plain HTML, CSS and JavaScript to keep it simple. In the future I might add some libraries or frameworks if needed.

## Project

### Drag & Drop

This project is a simple drag and drop implementation using the HTML5 drag and drop API.

It allows you to drag elements from one list to another. The dragged element will be removed from the source list and added to the target list.

## Backend

This repository will have a simple backend implemented in Node.js using Express. It will be used to serve the frontend files and handle any API requests if needed.
